教你本地安裝/運行 Z-Image Turbo 最簡單的方法

1. 下載 ComfyUI

從 GitHub 下載 ComfyUI v0.18.2 官方版，直接訪問這個鏈接：
https://github.com/Comfy-Org/ComfyUI/releases/tag/v0.18.2

下載檔名 適用顯示卡
ComfyUI_windows_portable_amd.7z AMD 顯示卡
ComfyUI_windows_portable_nvidia.7z NVIDIA 顯示卡（20/30/40 系列）
ComfyUI_windows_portable_nvidia_cu126.7z NVIDIA 顯示卡（推薦，50 系列）
2. 解壓縮（關鍵：路徑不能有中文 / 空格）

路徑範例（正確）：
D:\CUI

❌ 錯誤範例：D:\CUI\繪圖軟體\ComfyUI（包含中文會導致啟動失敗）

3. 下載 Z-Image Turbo 模型

請到 ComfyUI 說明頁面下載：
https://docs.comfy.org/tutorials/image/z-image/z-image-turbo

下載後將 .safetensors 模型檔放到對應資料夾：

📂 ComfyUI/
├── 📂 models/
│　　├── 📂 text_encoders/
│　　│　　　└── qwen_3_4b.safetensors
│　　├── 📂 diffusion_models/
│　　│　　　└── z_image_turbo_bf16.safetensors
│　　└── 📂 vae/
│　　　　　　└── ae.safetensors

4. 啟動 ComfyUI

進入解壓縮目錄：

  • NVIDIA 顯示卡：雙擊 run_nvidia_gpu.bat
  • CPU 模式：雙擊 run_cpu.bat（不建議，速度極慢）

首次啟動：
會彈出黑框（命令列），自動下載依賴（約 1–3 分鐘）
完成後瀏覽器自動開啟：http://127.0.0.1:8188

進入後點擊「模版」→ 選擇「文生圖」即可開始使用 Z-Image Turbo。

下 Prompt (可中英文)，然後 "執行"

例如 Prompt： 18歲女學生成群結隊去買鹹酥雞，攤位名稱"老張鹹酥雞"，內容字體必須是繁體中文。
